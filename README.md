# HCI HW4

人機互動作業4，有左右2個熱區使用背景鄉計算現在畫面是否異動。
當右側熱區偵測到異動值過大時(e.g. 舉起手到熱區)，則會進行拍照
當左側熱區偵測到異動值過大時(e.g. 舉起手到熱區)，則會顯示最新拍下的照片

## Installation

```bash
pip3 install python-opencv
```

```bash
python3 main.py
```
